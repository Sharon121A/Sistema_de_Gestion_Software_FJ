# Sistema_de_Gestion_Software_FJ

## Descripción
Sistema desarrollado en Python que gestiona clientes, servicios y reservas
sin uso de bases de datos, aplicando Programación Orientada a Objetos y
manejo avanzado de excepciones.

## Características
- Abstracción, herencia, encapsulación y polimorfismo
- Excepciones personalizadas
- Registro de errores en archivo log
- Ejecución secuencial y estable ante errores

## Ejecución
Ejecutar el archivo principal:
python main.py
